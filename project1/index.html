SYMBOLRT (SYmbolic Model-Based test case generation toOL for Real-Time systems) is a model-based test generation tool developed to automatically generate test cases from symbolic models in order to test real-time systems.

SYMBOLRT is open source software and is distributed under the GNU GPL license (version 3).
